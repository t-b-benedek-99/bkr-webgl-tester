BOOKR Embed player integration demo

Example code to embed BOOKR player and capture the reading & excercise statistics with JS.

Use the following embed code to display BOOKR books in your website: 
<iframe src="https://d35kui49fsydak.cloudfront.net/?book=517" title="book" ></iframe>
Replace the book parameter with the ID of the book you would like to embed.

RAW statistic data

Capture the RAW statistic data with the following JS script. This example prints the captured RAW statistic data to the console.

CODE


<script>
window.onmessage = function(e) { 
	console.log(e.data);
	//YOUR CODE HERE
}
</script>

RESULT

This code captures the raw statistic data on book close and page flip on games.


Response format (reading stat):

{
	"bookId":"517",
	"startedAt":"2022-02-15T08:08:29.9390000+01:00",
	"duration":4.06489896774292,
	"pagesVisited":[0,1,2],
	"lastPageVisited":2,
	"userId":""
}
  • bookId: the ID of the current book
  • startedAt: The timestamp when the read session is started in datetime format
  • duration: The time the user spent reading the book in the current session, in seconds
  • pagesVisited: The list of pages the user visited in the current session
  • lastPageVisited: The last page the user visited in the current session.

Response format (game stat):

{
	"mUserId":"1",
	"mBookId":"follo2100370",
	"mGameTypeId":"draganddrop",
	"mPageNumber":9,
	"mPageGuid":"90545d16-0848-4a19-864d-037ba0f51fb1",
	"mStatus":false,
	"mNumberOfElements":6,
	"mNumberOfCorrectAttempts":0,
	"mNumberOfWrongAttempts":0
	}
  • mGameTypeId: The type of the game
  • mPageNumber: The page number where the game is located in the book
  • mStatus: true if the user completed the game, else false
  • mNumberOfElements: Number of all elements the game consist of. For example the number of questions.
  • mNumberOfCorrectAttempts: The number of correct answers.
  • mNumberOfWrongAttempts: The number of incorrect answers.
